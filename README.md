# Visualizador de series temporales de vistas de página

Proyecto: visualizaciones de la serie temporal de vistas diarias del foro de freeCodeCamp.

Archivos principales:
- `time_series_visualizer.py` : funciones draw_line_plot, draw_bar_plot, draw_box_plot
- `main.py` : script para generar las imágenes
- `fcc-forum-pageviews.csv` : dataset (subir en la raíz)

Requisitos:
`pip install -r requirements.txt`

Ejecutar:
`python main.py`

Las imágenes se guardarán como:
- `line_plot.png`
- `bar_plot.png`
- `box_plot.png`
